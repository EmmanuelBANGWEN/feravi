Documentation - Feravi stat

Application de suivi climatique et de production d'œufs pour les fermes avicoles

1. Objectif du projet

Le projet Feravi stat a été conçu pour aider les éleveurs de poules pondeuses à mieux comprendre l’impact des conditions climatiques sur la production d’œufs. L’objectif est de faciliter la prise de décision grâce à une meilleure lecture des données environnementales et de production.

2. Source d’inspiration

Ce projet est né d’un constat sur le terrain : de nombreux éleveurs ne disposent pas d’outils simples et accessibles pour faire le lien entre les conditions de leur élevage et la productivité de leurs animaux. Cette application répond donc à un besoin réel de suivi quotidien et d’analyse sans nécessiter de connaissances informatiques avancées.

3. Comment fonctionne l’application ?

L’utilisateur enregistre chaque jour les conditions de température et d’humidité à trois moments précis (08h, 13h et 17h) ainsi que le nombre d’œufs pondus. Ces données sont stockées dans une base et visualisées sous forme de tableaux et graphiques dynamiques.

  • Relevé manuel des données par l’éleveur
  • Saisie via l’interface web
  • Visualisation automatique via des graphiques (Plotly)

4. Résultats attendus

  • Amélioration de la productivité par l’ajustement de l’environnement
  • Découverte de tendances ou corrélations (ex. humidité élevée = baisse de production)
  • Optimisation des pratiques d’élevage sur base de données concrètes

7. Étapes d'utilisation

- Collecte des données

Relever les informations suivantes trois fois par jour (08h, 13h, 17h) :

  • Température (°C)
  • Humidité (%)
  • Nombre d’œufs pondus
  • Nombre de poules

- Enregistrement

Rendez-vous dans le formulaire de saisie pour entrer les relevés de la journée.

- Visualisation

Les données sont converties en graphiques lisibles depuis le tableau de bord.

8. Explication des graphiques

Chaque graphique est conçu pour être compréhensible même pour un non-informaticien. Voici ce que vous pouvez observer :

- Évolution de la Température

Visualisez la variation de température à 08h, 13h et 17h. Permet d’identifier les jours trop chauds ou froids.

- Évolution de l’Humidité

Suivi des taux d’humidité sur les mêmes créneaux horaires pour évaluer la qualité de l’air ambiant.

- Production quotidienne d’œufs

Affiche le nombre total d’œufs pondus chaque jour. Aide à détecter des baisses ou pics de production.

- Corrélation humidité / production

Met en évidence s’il existe un lien direct entre humidité et production d’œufs.

- Corrélation température / humidité

Compare les deux paramètres pour voir s’ils varient ensemble ou non.

Comment lire les graphiques :

  • Pointez sur les points pour voir les valeurs exactes.
  • Utilisez les menus déroulants pour filtrer par date ou heure.
  • Comparez visuellement plusieurs jours pour repérer les anomalies ou tendances.

9. Filtres et analyses personnalisées

Utilisez les filtres disponibles dans le tableau de bord pour :

  • Comparer plusieurs jours
  • Zoomer sur une période précise
  • Observer un moment particulier de la journée
