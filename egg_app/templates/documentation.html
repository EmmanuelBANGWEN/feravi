Documentation - Farm Monitoring

Suivi de la température, l’humidité et la production d'œufs

1. Introduction

L’application Farm Monitoring permet d’enregistrer et d’analyser les conditions climatiques de la ferme.

  • Enregistrement des relevés climatiques et de ponte.
  • Affichage des données sous forme de tableau.
  • Visualisation des tendances avec des graphiques interactifs.
  • Exportation des données.

2. Technologies Utilisées

ComposantTechnologie
Back-endDjango
Base de donnéesSQLite
Front-endBootstrap, HTML, CSS
VisualisationPlotly, Pandas

3. Installation

4. Modèle de Données

class Measurement(models.Model):
    date = models.DateField(auto_now_add=True)
    temperature = models.FloatField()
    humidity = models.FloatField()
    egg_count = models.IntegerField()
