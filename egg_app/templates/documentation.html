Documentation - Feravi stat

Guide d'utilisation de l'application de suivi de la production de poules pondeuses

1. Introduction

L’application Feravi stat permet d’enregistrer et d’analyser les conditions climatiques de la ferme.

  • Enregistrement des relevés climatiques et de ponte.
  • Affichage des données sous forme de tableau.
  • Visualisation des tendances avec des graphiques interactifs.
  • Exportation des données.

2. Technologies utilisées

ComposantTechnologie
Back-endDjango
Base de donnéesSQLite
Front-endBootstrap, HTML, CSS
VisualisationPlotly, Pandas

3. Installation

git clone <repo_url>
cd egg
python -m venv venv
Windows : venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver

Étapes d'utilisation

1. Collecte des données

Utilisez un hydrothermomètre pour mesurer :

  • La température
  • L’humidité

Mesures à prendre à :

  • 08:00
  • 13:00
  • 17:00

2. Enregistrement des données

  • Date
  • Créneau horaire
  • Température
  • Humidité
  • Nombre d’œufs
  • Nombre de poules

3. Visualisation

Accédez à des graphiques interactifs depuis le tableau de bord.

Explication des graphiques

Voici les différents graphiques que vous pouvez consulter. Ils vous aident à mieux comprendre l’état de votre élevage jour après jour.

Température au fil des jours

Ce graphique montre comment la température évolue chaque jour à trois moments clés : matin (08h), après-midi (13h) et soir (17h). Il aide à détecter les périodes trop chaudes ou trop froides.

Humidité au fil des jours

Affiche les niveaux d’humidité à différents moments de la journée. Utile pour vérifier si l’environnement est trop sec ou trop humide.

Production d’œufs par jour

Montre combien d’œufs sont pondus chaque jour. Cela aide à suivre la productivité de vos poules.

Lien entre humidité et production

Compare les niveaux d’humidité avec le nombre d’œufs produits. Cela peut révéler un lien entre les deux.

Lien entre température et humidité

Permet de voir si température et humidité sont élevées en même temps, ce qui pourrait influencer les conditions de l’élevage.

Astuce : Utilisez les filtres en haut de la page pour cibler une période ou un moment spécifique.

Filtrer les données

Vous pouvez sélectionner une période ou un moment de la journée spécifique pour analyser les données.

À qui s’adresse l’application ?

  • Aux éleveurs qui souhaitent suivre et améliorer la production.
  • À toute personne voulant comprendre les conditions de son élevage.

Exemple d’interprétation

"Ce matin, j’ai mesuré 28°C et 65% d’humidité. Je vais maintenir ce niveau d’humidité."

Résumé technique simplifié

  • Ce que vous enregistrez : température, humidité, œufs, nombre de poules
  • Ce que vous voyez : tableaux, graphiques interactifs
  • Fonctionnalités : filtres, exportation, vue quotidienne
