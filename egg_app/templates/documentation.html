Documentation - Feravi stat

Guide d'utilisation de l'application de suivi de la production de poules pondeuses

1. Introduction

L’application Feravi stat permet d’enregistrer et d’analyser les conditions climatiques de la ferme.

  • Enregistrement des relevés climatiques et de ponte.
  • Affichage des données sous forme de tableau.
  • Visualisation des tendances avec des graphiques interactifs.
  • Exportation des données.

2. Technologies Utilisées

ComposantTechnologie
Back-endDjango
Base de donnéesSQLite
Front-endBootstrap, HTML, CSS
VisualisationPlotly, Pandas

3. Installation

git clone <repo_url>
cd egg
python -m venv venv
Windows : venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver

Étapes d'utilisation

1. Collecte des données

Utilisez un hydrothermomètre pour mesurer :

  • La température
  • L’humidité

Mesures à prendre à :

  • 08:00
  • 13:00
  • 17:00

2. Enregistrement des données

  • Date
  • Créneau horaire
  • Température
  • Humidité
  • Nombre d’œufs
  • Nombre de poules

3. Visualisation

Accédez à des graphiques interactifs depuis le tableau de bord.

📊 Explication des Graphiques

Voici les différents graphiques que vous pouvez consulter sur cette page. Ils vous aident à mieux comprendre l’état de votre élevage jour après jour.

🌡️ Température au fil des jours

Ce graphique montre comment la température a changé chaque jour, à trois moments de la journée : le matin (08h), l’après-midi (13h) et le soir (17h). Cela vous permet de voir s’il fait trop chaud ou trop froid pour vos poules.

💧 Humidité au fil des jours

Ce graphique affiche le niveau d’humidité chaque jour, aussi à 08h, 13h et 17h. Vous pouvez savoir si l’air est trop sec ou trop humide, ce qui peut influencer le bien-être des poules.

🥚 Production d’œufs par jour

Ce graphique montre combien d’œufs ont été pondus chaque jour. Il est utile pour suivre la production et repérer les baisses ou les hausses.

💧 ➡️ 🥚 Humidité et œufs

Ce graphique permet de voir s’il y a un lien entre le niveau d’humidité et le nombre d’œufs pondus. Par exemple, vous pourriez remarquer que les poules pondent plus quand l’air est un peu plus humide.

🌡️ ➡️ 💧 Température et humidité

Ce graphique compare la température et l’humidité au même moment. Il permet de savoir s’il fait chaud et humide en même temps, ce qui peut avoir un impact sur les poules et la production.

Astuce : Utilisez les filtres en haut pour choisir une période précise ou un moment de la journée, et ainsi mieux comprendre ce qui se passe dans votre élevage.
